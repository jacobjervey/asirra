# Asirra

This is a simple gem that assists you in implementing the ASIRRA CAPTCHA system from Microsoft into your Rails application.

## Installation

Add this line to your application's Gemfile:

    gem 'asirra'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install asirra
